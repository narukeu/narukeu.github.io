Babel 是如何处理箭头函数的？

Babel 是一个 JavaScript 编译器，它可以把新版本的 JavaScript 代码转换成兼容旧环境的代码。其中一个常用的转换功能是把箭头函数转换成普通函数，但是仍然保持了箭头函数的词法作用域。本文将从 Babel 的特性出发，来介绍 Babel 转换箭头函数的原理。
