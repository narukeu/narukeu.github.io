在主机上设置 Linux 客户机 SSH 免密钥登录

1. 生成 SSH 密钥对

Section titled “1. 生成 SSH 密钥对”

ed25519 是现代且安全的算法，而且比 RSA 要短。也可以用传统的 RSA。

然后建议最好设置一个口令。

2. 将公钥复制到 Linux 客户机

Section titled “2. 将公钥复制到 Linux 客户机”

SSH 登录客户机，然后用 nano 编辑 ~/.ssh/authorized_keys 文件，将公钥（*.pub）内容复制进去保存退出。

然后设置 600 权限。
