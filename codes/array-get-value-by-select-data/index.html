注意事项

Section titled “注意事项”
  • 匹配使用严格相等（===），请确保传入 input 的类型与数据源中的 value 类型一致。
  • 返回值始终为字符串；如果原始 label 不是字符串，函数会调用 String(…) 转换。
  • 对于大型数组，find 是 O(n) 的线性查找；若频繁查询可考虑先构建 Map 索引以提升性能。
